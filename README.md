# Instructions 
to run: 
```
cd ./accucook
npm install
npm run start 
```
## Technologies used:
* Javascript ES6
* Webpack 4.43.0
* Babel-polyfill 6.26.0
* Node.js
* [Forkify API](https://forkify-api.herokuapp.com/)
* [Fraction] (https://github.com/ekg/fraction.js/)
* [Uniqid](https://github.com/adamhalasz/uniqid)
